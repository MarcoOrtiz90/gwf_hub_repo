The Parser tool allows the user to input the name of an existing flow in Albacore, so that the tool can fetch for the JSON codes and convert them into a format ready Source File.How to use the tool:

  1. Enter the workflow name in the empty field.
  2. Select the region in which the workflow is located.
  3. Select your default browser. The chosen browser has the be the regular used browser for accesing Amazon related websites.
  4. Once settings are done, click the GENERATE! button to create the Source File based on the workflow id provided
  5. The application will start the automated process of accesing Albacore, pulling the flow and its respective JSON codes to create the Source File.
  6. Once finished a folder is created on the user's desktop with the requested Source File.
